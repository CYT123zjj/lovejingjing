案例2:tar制作/释放归档压缩包（zcf、ztf、zxf、jcf、jtf、jxf、cf、tf）
  首先创建/root/boothome/与/root/usrsbin/目录
    1）备份/boot、/home这两个文件夹，保存为boothome.tar.gz文件
[root@localhost ~]# tar -zcf boothome.tar.gz /boot /home
    2）查看boothome.tar.gz文件内包含哪些内容 
[root@localhost ~]# tar -tf boothome.tar.gz
    3）将boothome.tar.gz释放到文件夹/root/boothome/下
[root@localhost ~]# tar -xf boothome.tar.gz -C /root/boothome
    4）备份/usr/sbin目录，保存为usrsbin.tar.bz2文件
[root@localhost ~]# tar -jcf usrsbin.tar.bz2 -C /usr/sbin/
    5）查看usrsbin.tar.bz2文件内包含哪些内容
[root@localhost ~]# tar -tf usrsbin.tar.bz2
    6）将usrsbin.tar.bz2释放到/root/usrsbin/文件夹下
[root@localhost ~]# tar -xf usrsbin.tar.bz2 -C /root/usrsbin/

案例3：添加并测试用户账号
    1）创建一个名为stu01的用户账号
[root@localhost ~]# useradd stu01
    2）检查/etc/passwd、/etc/shadow文件的最后一行
tail -l /etc/passwd    tail -l /etc/shadow
    3）检查/home/新增加的宿主目录（家目录）
[root@localhost ~]# cd /home
[root@localhost home]# pwd
    4）为用户stu01设置一个密码（123456）
[root@localhost home]# passwd stu01
案例4：添加账号时设置不同属性
    1）新建用户nsd01，宿主目录位于/opt/nsd01
[root@localhost home]# useradd -d /opt/nsd01  nsd01
    2）新建系统账号sys01，将UID设为1234，登录Shell设为/sbin/nologin
[root@localhost home]# useradd -u 1234 -s /sbin/nologin  shell
    3）为用户sys01设置密码，并测试是否能够登录
[root@localhost home]# passwd sys01
    4）新建用户admin，附加组设为adm、root
usermod -G adm root
usermod -G root adm

案例7：组账号基本管理
    1）新建组账号stugrp
[root@server0 ~]# groupadd stugrp
    2）为stugrp组添加三个成员用户（user01、root、zhangsan）
[root@server0 ~]# useradd user01 
	[root@server0 ~]# useradd root
	[root@server0 ~]# useradd zhangsan
	[root@server0 ~]# gpasswd -a user01 stugrp
	[root@server0 ~]# gpasswd -a root stugrp
	[root@server0 ~]# gpasswd -a zhangsan stugrp
    3）从stugrp组删除一个成员（user01）
[root@server0 ~]# gpasswd  -d user01 stugrp
